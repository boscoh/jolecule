Keyboard Shortcuts

A - add atom label

S - show all sidechains
C - hide all sidechains
N - show neighbour sidechains around central atom

B - cycle through backbone representations
H - hide/show hydrogens
W - hide/show waters
L - hide/show ligands

[numbers] - choose view
J/K - center to prev/next residue
[up]/[down] - select prev/next view
' ' - next view


Widget Controls

j - got to full page version
lig - hide/show ligands
h - hide/show hydrogen atoms
wat - hide/show water molecules

bb - cycle through backbone styles

sc - show all sidechains
clr - hide all sidechains
neig - show neighbouring sidechains of central residue

loop - start/stop slideshow
create - make new view
text - show/hide view text
